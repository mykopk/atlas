# Changelog

## 2.0.0 (2026-07-04)

### Initial Release

- Multi-adapter support: Drizzle ORM, Prisma, SQL, Supabase, Mock
- Extension decorator chain: Encryption, SoftDelete, Caching, Audit, ReadReplica, MultiWrite, MultiRead
- NestJS integration with `AtlasModule.forRoot()` / `forRootAsync()`
- Repository pattern with `BaseRepository<T>` and QueryBuilder
- Advanced features: Sharding, Multi-tenancy, Connection Pooling, Monitoring, Backup
- Migration and Seed management
- Security sanitizers and validation pipes
- Dual CJS/ESM build with TypeScript declarations
- Comprehensive JSDoc documentation
